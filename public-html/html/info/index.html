Informació del Projecte La Cantina

Descripció

La Cantina és un projecte web desenvolupat amb Apache i Docker per aprendre sobre virtualització i configuració de servidors web.

Tecnologies utilitzades

  • Docker
  • Apache HTTP Server
  • VirtualHosts
  • HTML/CSS

Objectius

  • Configurar múltiples VirtualHosts
  • Gestionar DocumentRoot diferents
  • Treballar amb logs separats
